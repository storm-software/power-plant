# Run Power Plant

This section provides instructions on how to run Power Plant and introduces some
notable command-line options available to customize its operation.

```admonish
Content in this section is currently under development. Please check back later for updates.
```

**To start Power Plant after installation:**

```bash
power-plant run
```
